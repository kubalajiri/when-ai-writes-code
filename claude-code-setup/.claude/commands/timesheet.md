# Timesheet Generator

Generate a Git activity timesheet for the specified month.

**Usage:** `/timesheet [month]` (e.g., `/timesheet november 2025` or `/timesheet 2025-11`)

## Arguments
- $ARGUMENTS: Month to generate (format: "YYYY-MM", "november 2025", "11 2025", or empty for current month)

## Instructions

Run the timesheet script and output the result:

```bash
python3 ~/.claude/scripts/timesheet.py $ARGUMENTS
```

Output the script result as plain text. Do not add any formatting or commentary.
